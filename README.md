#  Color Picker App

This project is a modern **Flutter** application compatible with both **Web** and **Mobile**, allowing users to explore different color tones, generate random colors, and view color codes.

With its user-friendly interface, you can learn color names or copy their codes for use in your designs.

## Features

* **Extensive Color Palette:** Includes standard Material colors as well as custom tones (Gold, Salmon, Mint, etc.).
* **Random Mode:** Discover surprise colors with the "Random" button.
* **Dynamic Theme:** The app's general theme and background change instantly based on the selected color.
* **Shape Switching:** View the color palette as either a square or a circle.
* **Web & Mobile Compatible:** Works on iOS, Android, and Web browsers with a single codebase.

## Installation & Setup

flutter pub get

# Run the app
flutter run
